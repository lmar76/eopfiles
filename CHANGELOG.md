# Changelog

## [0.2] - 2023-02-16

### Changed
- Changed `validity_start` and `validity_stop` type from `Optional[str]` to
  `datetime.datetime`.

## [0.1] - 2023-02-15

### Added
- Support for Restituted Orbit File and Orbit State Vector file (`AUX_ORBREF`).
- Support for Predicted orbit File (`MPL_ORBPRE`).
